Overview
--------

PyAGW3 provides a full-featured, callback-driven client for connecting to AGWPE-compatible packet radio engines (e.g., original AGWPE, Dire Wolf, UZ7HO SoundModem).

Key Features
~~~~~~~~~~~~
- Exponential backoff connection with TCP keepalive
- Unproto, connected, and raw frame modes
- Monitoring, heard stations, outstanding frames
- Extended version and memory queries
- Thread-safe with comprehensive error handling

See the :ref:`api` for full class reference.
